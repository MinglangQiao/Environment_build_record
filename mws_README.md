## Usage
### Test

0. Environment: python2.7, pytorch'0.5.0a0+54db14e'

1. [Download models](https://1drv.ms/f/s!AqVkBGUQ01XGhyeDYsaaxvAZXW7k) and put in the current folder



1. Run

```bash
python main.py \
--img_dir 'path/to/images(.jpg)' \
--gt_dir 'path/to/ground-truth(.png)'

```
